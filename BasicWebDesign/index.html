Transitions and Animations Review

What does the CSS transition property do?
It creates smooth changes between property values over a set time.
What two values are needed for a basic transition?
The property to change and the duration of the change.
What CSS property changes the width over 3 seconds?
transition: width 3s;
How do you trigger a transition in CSS without JavaScript?
Use the :hover pseudo-class to change the property value.
Whatâ€™s the difference between transition and animation?
Transitions go from one state to another; animations can have multiple steps.
What keyword is used to define CSS animations?
@keyframes
What does "from" and "to" represent in @keyframes?
They represent the start (0%) and end (100%) of an animation.
How do you apply an animation to an element?
Use properties like animation-name and animation-duration.
